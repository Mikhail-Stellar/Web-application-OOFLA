2 типа пользователя: гость, авторизованный (файлы reg).

Основная часть реализована на языке PHP (вёрстке и дизайну уделена малая часть, упор на backend).
Существует 2 БД: registration.sql (для реализации аутентификации и хранения данных о пользователях с шифрованием паролей), Kursovoy.sql (основная БД из которой идет выгрузка данных в веб-приложение, а также все отправленные сообщения через формы от пользователей).
Папка Content.rar необходима для хранения файлов и изображений.
